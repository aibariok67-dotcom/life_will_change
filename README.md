# Persona 5 V-Gesture Music Player 🎶🎭

Этот проект использует вебкамеру и библиотеку **MediaPipe** для распознавания жеста "V" (знак мира / Peace sign). При обнаружении жеста автоматически запускается легендарный трек **"Life Will Change"** из игры *Persona 5 Royal*.

Проект имеет **два варианта реализации**: локальный скрипт на Python и веб-версию для браузера.

---

## 🌐 Вариант 1: Веб-версия (Для запуска в браузере и отправки друзьям)
Этот вариант не требует установки Python на ПК пользователя и идеально подходит для отправки друзьям на Windows. Она открывает стильный интерфейс в стиле Persona 5 с веб-камерой и встроенным YouTube-плеером на одной странице.

### Локальное тестирование:
Современные браузеры блокируют веб-камеру для локальных файлов (`file://`). Чтобы запустить веб-страницу локально на сервере, выполните:
```bash
python server.py
```
*(Скрипт автоматически запустит локальный сервер на порту `8080` и откроет страницу в браузере)*.

### Размещение в интернете:
Чтобы друг мог открыть проект в один клик без консоли:
1. Выложите этот репозиторий на **GitHub**.
2. Включите **GitHub Pages** в настройках репозитория (Settings -> Pages -> Build and deployment -> Source: Deploy from a branch -> Выберите `main` ветку).
3. Дайте другу ссылку вида `https://ваше_имя.github.io/имя_репозитория/` — проект заработает на любом устройстве!

## 🐍 Вариант 2: Локальный Python-скрипт
Классическое десктопное приложение на Python.

### Установка зависимостей:
В папке проекта выполните (рекомендуется использовать виртуальное окружение `venv`):
```bash
pip install -r requirements.txt
```

### Запуск:
```bash
# Использовать автоопределение камеры (с пропуском Loopback-устройств):
python main.py

# Или явно указать индекс камеры (например, 1):
python main.py 1
```

### Управление в Python-версии:
- **Жест V:** Держите руку вертикально перед камерой.
- **Q / ESC:** Нажмите `q` или `Esc` в окне видеопотока, чтобы выйти из программы.
